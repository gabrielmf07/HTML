Inserseccion de imagenes

Para insertar una imagen usamos la etiqueta 'img' y usamos principalmente los atributos alt='' para darle unos valores extra a esa etiqueta, con el alt decimos que en caso no se muestra la imagen se muestre el mensaje que pusimo dentro de esta. Tambien, tenemos otro atributo src='' indicamo el contenido de la imagen ya sea una url o una carga de imangen del proyecto individual.

Ademas, podemos asignarle un ancho y un alto a nuestra imagen usando width y height despues del img.

Como podemos ver, en la segunda img se uso un height, para equilibrarlas con las demas.
